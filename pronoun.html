Pronunciation (Pengucapan)

1. Voiced & Voiceless Sound

A. Voiced Sound (Bunyi Getar)

Saat diucapkan, tenggorokan bergetar.

[ a, b, d, g, h, j, l, m, n, o, r, u, v, w, y, z, ng ]
  • bring → akhir ng → voiced
  • go → akhir o → voiced
  • boy → akhir y → voiced

B. Voiceless Sound (Bunyi Tidak Getar)

Saat diucapkan, tenggorokan tidak bergetar.

[ ch, f, k, p, q, s, sh, t ]
  • book → akhir k → voiceless
  • think → akhir k → voiceless
  • teach → akhir ch → voiceless

2. Akhiran S / ES

voiced → [z]
voiceless → [s]

Jika Akhirannya Voiced → Dibaca [z]

bring → brings = bringz
win → wins = winz
boy → boys = boyz

Jika Akhirannya Voiceless → Dibaca [s]

walk → walks
book → books
think → thinks
